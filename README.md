# QA AI Testing Lab

Projeto de QA utilizando Inteligência Artificial.

## Objetivo

Criar um laboratório completo de testes de software utilizando:

- QA Manual
- Inteligência Artificial
- Automação de Testes
- Testes de API
- Documentação

- ChatGPT
- Cypress

- Postman
- Git
- GitHub
